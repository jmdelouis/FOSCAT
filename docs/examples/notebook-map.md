# Notebook map

Use the notebooks from `demo-foscat-pangeo-eosc` as the reference examples.

| Notebook | Documentation page | Description |
|---|---|---|
| `Demo_Synthesis.ipynb` | HEALPix synthesis | Global spherical synthesis from scattering-covariance statistics |
| `Synthesis2D.ipynb` | 2D synthesis | Planar image/field synthesis |
| `Denoising-2D.ipynb` | 2D synthesis | Denoising by statistical constraints |
| `Remove_CMB.ipynb` | Component separation | Separating or removing a CMB-like component |
| `local_foscat.ipynb` | Local HEALPix wavelets | Local domains and incomplete spherical maps |
| `CNN_local.ipynb` | HEALPix neural networks | Local convolutional model on HEALPix |
| `CNN_ecmwf.ipynb` | HEALPix neural networks | Example on ECMWF-like fields |

## Documentation policy

Do not link users to the older `FOSCAT_DEMO` repository as the primary documentation source. Keep examples close to maintained notebooks and copy minimal, tested snippets into the docs.
